# 📖 About

Firearms Monitoring System is an AI-powered web application that detects firearms from uploaded images using Machine Learning and Computer Vision.

The system performs:

- Image Upload
- Image Preprocessing
- Feature Extraction
- Model Prediction
- Firearm Classification
- Prediction Confidence
- Detection History

# ⚙ Installation

Clone Repository

```bash
git clone https://github.com/Narender1413/Firearms-Monitoring-System.git
```

Move into Folder

```bash
cd Firearms-Monitoring-System
```

Install Packages

```bash
pip install -r requirements.txt
```

Run Application

```bash
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```

---

# 🚀 Workflow

1. User opens application.
2. Uploads firearm image.
3. Image is preprocessed.
4. ML model extracts features.
5. Prediction is generated.
6. Result and confidence score are displayed.
7. Detection history is stored.
